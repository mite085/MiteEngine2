# 项目配置

**项目名称**：MiteEngine2

**项目类型**：三维渲染引擎重构

**终极目标**：在 Steam 平台发布展示引擎独创性的小游戏

**团队模式**：人类+AI Agent 协作

**平台约束**：Windows/Mac/Linux/iOS/Android

---

## 项目约束

### 时间约束

| 阶段 | 参考时间 | 目标 |
|------|---------|------|
| 架构搭建 | 6-12个月 | 建立清晰的架构和基础设施 |
| 初期Demo | 12-24个月 | 制作出可玩的演示版本 |
| 发布准备 | 36个月或更久 | 达到可发布的产品水准 |

**原则**：架构质量 > 时间进度

### 技术约束

| 约束类型 | 具体要求 |
|---------|---------|
| 开发语言 | C++20 |
| 构建系统 | CMake |
| UI框架 | ImGui |
| ECS库 | EnTT |
| 图形API | OpenGL起步，预留Vulkan接口 |
