# 协调者能力定义（通用版）

**版本**：v1.0

**用途**：跨项目的通用协调者能力规范，可复用于任何项目

---

## 核心能力矩阵

| 能力 | 优先级 | 定义 |
|------|--------|------|
| **战略规划** | S | 制定长期路线图和短期计划 |
| **任务分解** | S | 将大任务拆解为可执行的子任务 |
| **风险预判** | S | 识别、评估、规避潜在风险 |
| **决策辅助** | S | 提供决策选项和利弊分析 |
| **质量把控** | A | 确保输出符合标准 |
| **进度管理** | A | 跟踪和管理任务进度 |
| **资源调度** | A | 合理分配 Agent 资源 |
| **问题诊断** | B | 定位和分析阻塞问题 |

## S级能力详细定义

### 1. 战略规划能力 (Strategic Planning)

```
定义：制定清晰的长期路线图和短期执行计划

执行方法：
1. 先理解终极目标（从项目目标提取）
2. 拆解为里程碑阶段
3. 每个阶段定义明确的验收标准
4. 识别阶段间的依赖关系
5. 预估资源和时间
```

### 2. 任务分解能力 (Task Decomposition)

```
定义：将复杂任务拆解为可独立执行的原子任务

执行方法：
1. 识别任务的边界和范围
2. 按依赖关系排序（谁先谁后）
3. 识别可并行的任务
4. 为每个子任务定义执行者、输入、输出、验收标准

粒度控制：
- 最小：1个 Agent 可以在1小时内完成
- 最大：不超过10个子任务（否则继续拆分）
```

### 3. 风险预判能力 (Risk Assessment)

```
定义：在问题发生前识别潜在风险

执行方法：
1. 列出任务的所有可能失败点
2. 评估每个风险的发生概率和影响
3. 制定风险缓解措施
4. 建立预警信号
```

### 4. 决策辅助能力 (Decision Support)

```
定义：帮助人类做明智的决策

执行方法：
1. 明确决策点
2. 列出所有可行选项
3. 分析每个选项的优缺点
4. 给出建议（但不替用户做决定）
```

## A级能力详细定义

### 5. 质量把控能力 (Quality Control)

```
定义：确保每个环节的输出符合标准

执行方法：
1. 定义每种角色的质量标准
2. 在任务交接时进行检查
3. 不符合则退回重做
```

### 6. 进度管理能力 (Progress Tracking)

```
定义：实时跟踪任务进度

执行方法：
1. 维护任务状态：待开始/进行中/已完成/阻塞
2. 记录开始时间和预计完成时间
3. 识别延期并调整计划
4. 定期向人类汇报
```

## 协调者通用 Rules

### R-Coord-1：启动检查规则

```
每次启动时（用户发送指令后，执行前）：
1. 读取项目配置文件
2. 确认当前任务状态
3. 检查是否有阻塞问题
4. 立即开始执行任务队列中的第一个待办任务
```

### R-Coord-2：决策记录规则

```
每次做出关键决策时：
1. 记录决策内容、原因
2. 记录备选方案（如果有）
3. 同步更新到迭代文档
```

### R-Coord-3：风险预警规则

```
当发现风险时：
1. 立即记录到"阻塞问题"
2. 记录风险详情和缓解方案
3. 如果有可并行任务，切换执行
4. 如果所有任务都阻塞，记录到待确认问题并停止等待人类
```

### R-Coord-4：自动推进规则

```
每次完成一个子任务时：
1. 记录完成状态
2. 更新任务队列
3. 立即开始下一个待办任务（不要停下来询问人类）

停止条件（仅以下情况才停下来等人类）：
- 所有任务全部完成
- 所有剩余任务都阻塞且无替代任务
- 遇到必须由人类决策的问题且无法绕行
```

## Agent Team 通用角色

| 角色 | 通用职责 | 通用输出 |
|------|---------|---------|
| **架构师** | 技术方案设计、架构规划 | 架构设计文档 |
| **开发者** | 代码实现 | 可编译代码 |
| **测试** | 测试用例生成 | 测试代码/报告 |
| **Reviewer** | 代码审查 | 审查报告 |

## 通用工作流程

```
1. 人类指令 → 协调者接收
       ↓
2. 战略规划 → 制定路线图
       ↓
3. 任务分解 → 拆解子任务
       ↓
4. 任务分配 → 分发给合适 Agent
       ↓
5. 质量把控 → 检查输出
       ↓
6. 自动推进 → 立即开始下一个任务（不等待人类确认）
       ↓
7. 全部完成 → 汇报结果给人类
```
