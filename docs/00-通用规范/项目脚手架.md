# 项目脚手架模板（通用版）

**版本**：v1.0

**用途**：项目初始代码结构模板，可复用于任何项目

## 核心文件模板

### CMakeLists.txt（根目录）

```cmake
cmake_minimum_required(VERSION 3.16)
project(MyProject VERSION 1.0.0 LANGUAGES CXX)

# C++ 标准
set(CMAKE_CXX_STANDARD 20)
set(CMAKE_CXX_STANDARD_REQUIRED ON)

# 编译器选项
if(MSVC)
    add_compile_options(/W4 /permissive-)
else()
    add_compile_options(-Wall -Wextra -Wpedantic)
endif()

# 依赖
find_package(Threads REQUIRED)

# 子目录
add_subdirectory(src)
add_subdirectory(tests)

# 可执行文件
add_executable(${PROJECT_NAME} src/main.cpp)
target_link_libraries(${PROJECT_NAME} PRIVATE project_core)
```

### src/CMakeLists.txt

```cmake
# 源码库
add_library(project_core STATIC
    Core/Core.cpp
    Renderer/Renderer.cpp
)

target_include_directories(project_core PUBLIC
    ${CMAKE_SOURCE_DIR}/include
)
```

### main.cpp（最小可运行）

```cpp
#include <iostream>
#include <memory>
#include "project/Core/Core.hpp"

int main(int argc, char* argv[]) {
    std::cout << "Hello, Project!" << std::endl;

    auto core = std::make_unique<project::Core>();
    core->Initialize();

    std::cout << "Project initialized successfully." << std::endl;

    return 0;
}
```

|---------|---------|---------|
| 游戏引擎 | +Editor/, +Runtime/ | EnTT, ImGui, glm |

## 快速开始命令

```bash
# 创建构建目录
mkdir build && cd build

# 配置
cmake .. -DCMAKE_BUILD_TYPE=Release

# 编译
cmake --build . --config Release

# 运行
./MyProject
```

## 测试配置

### tests/CMakeLists.txt

```cmake
enable_testing()

    test_main.cpp
    test_core.cpp
)

target_link_libraries(test_project PRIVATE
    project_core
    doctest::doctest
)

add_test(NAME ProjectTests COMMAND test_project)
```
