# 项目配置模板

**用途**：每个新项目需要填写的配置信息，用于初始化协调者

---

## 项目基本信息

| 字段 | 说明 | 示例 |
|------|------|------|
| `项目名称` | 项目代号 | MiteEngine2 |
| `项目类型` | 软件类型 | 三维渲染引擎 |
| `终极目标` | 项目的最终交付物 | Steam平台小游戏 |
| `团队模式` | 开发协作方式 | 人类+AI Agent |
| `平台约束` | 目标运行平台 | Windows/Mac/Linux/iOS/Android |

---

## 使用方法

1. 创建新项目文件夹
2. 复制本模板为 `docs/项目配置.md`
3. 填写项目特定信息
4. 复制 `docs/00-通用规范/` 到新项目
5. 复制 `CLAUDE.md` 模板到新项目
6. 调整 CLAUDE.md 中的项目引用

---

**每个项目只需要填写本配置文件，即可复用通用协调者规范**
