# 平台层架构

**项目**：MiteEngine2 - SDF Raymarching 3D渲染引擎
**版本**：v1.0

## 1. 模块概述

### 1.1 平台模块定位

Platform模块是引擎的最底层，负责跨平台的系统抽象，包括窗口管理、输入处理、时间管理等。

### 1.2 子系统划分

```
Platform/
├── Window/      # 窗口管理
├── Input/       # 输入系统
├── Time/        # 时间管理
└── Device/      # 设备抽象
```

## 2. 窗口系统

### 2.1 窗口接口

```cpp
class IWindow {
public:
    virtual ~IWindow() = default;
    virtual void* GetNativeHandle() const = 0;
    virtual Vec2i GetSize() const = 0;
    virtual Vec2i GetPosition() const = 0;
    virtual void SetSize(int width, int height) = 0;
    virtual void SetTitle(const std::string& title) = 0;
    virtual void SetFullscreen(bool fullscreen) = 0;
    virtual bool IsFullscreen() const = 0;
    virtual void Show() = 0;
    virtual void Hide() = 0;
    virtual void Close() = 0;
    virtual bool ShouldClose() const = 0;
    virtual void SwapBuffers() = 0;
    virtual void PollEvents() = 0;
};
```

### 2.2 窗口事件

```cpp
enum class WindowEventType {
```

## 3. 输入系统

### 3.1 输入接口

```cpp
class IInputSystem {
public:
    // 键盘
    virtual bool IsKeyDown(KeyCode key) const = 0;
    virtual bool IsKeyPressed(KeyCode key) const = 0;
    virtual bool IsKeyReleased(KeyCode key) const = 0;

    // 鼠标
    virtual Vec2 GetMousePosition() const = 0;
    virtual Vec2 GetMouseDelta() const = 0;
    virtual bool IsMouseButtonDown(MouseButton button) const = 0;
    virtual float GetMouseWheel() const = 0;

    // 触控
    virtual int GetTouchCount() const = 0;
    virtual Touch GetTouch(int index) const = 0;
};
```
