# 资源管理模块架构

**项目**：MiteEngine2 - SDF Raymarching 3D渲染引擎
**版本**：v1.0
**角色**：模块架构师（Resources资源管理模块）
**日期**：2026年3月9日

---

## 1. 模块概述

### 1.1 Resources模块定位

Resources模块负责所有资源的加载、缓存、生命周期管理和序列化。

### 1.2 设计原则

| 原则 | 描述 |
|------|------|
| 异步优先 | 所有资源加载必须异步进行 |
| 引用计数 | 基于引用计数的自动资源释放 |
| 热重载 | 开发阶段支持资源热重载 |
| 依赖追踪 | 完整的资源依赖关系追踪 |

## 2. 资源系统

### 2.1 资源基类

```cpp
using AssetId = uint64_t;

enum class AssetType {
    Texture, Mesh, SDF, Material, Scene, Shader, Audio
};

class IAsset {
public:
    virtual ~IAsset() = default;
    AssetId GetId() const;
    AssetType GetType() const;
    uint32_t GetRefCount() const;
    void AddRef();
    uint32_t RemoveRef();

protected:
    virtual bool Load() = 0;
    virtual void Unload() = 0;
    virtual bool Reload() = 0;
};
```

### 2.2 资源句柄

```cpp
template<typename T>
class ResourceHandle {
public:
    T* Get() const { return asset_; }
    T* operator->() const { return asset_; }
    bool IsValid() const;
    AssetId GetId() const;

private:
    AssetId id_;
    T* asset_;
};

// 预定义句柄类型
using TextureHandle = ResourceHandle<Texture>;
using MeshHandle = ResourceHandle<Mesh>;
using SDFHandle = ResourceHandle<SDFAsset>;
using MaterialHandle = ResourceHandle<Material>;
using SceneHandle = ResourceHandle<Scene>;
```

## 3. 资源管理器

### 3.1 核心接口

```cpp
class IAssetManager {
public:
    template<typename T>
    ResourceHandle<T> Load(const std::string& path);

    template<typename T>
    void LoadAsync(const std::string& path, std::function<void(ResourceHandle<T>)> callback);

    virtual void Unload(AssetId id) = 0;
    virtual void Reload(AssetId id) = 0;
    virtual bool Exists(AssetId id) const = 0;
    virtual AssetMetadata GetMetadata(AssetId id) const = 0;
};
```

## 5. 序列化系统

### 5.1 序列化器接口

```cpp
class ISerializer {
public:
    virtual bool Serialize(IAsset* asset, const std::string& path, const SerializationContext& context) = 0;
    virtual std::unique_ptr<IAsset> Deserialize(const std::string& path, const SerializationContext& context) = 0;
};

class IAssetImporter {
public:
    virtual bool CanImport(const std::string& extension) const = 0;
    virtual std::unique_ptr<IAsset> Import(const std::string& sourcePath, const ImportOptions& options) = 0;
};
```
