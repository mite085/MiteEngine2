## 1. 模块概述

### 1.1 渲染模块定位

渲染模块是引擎的核心视觉输出层，负责将ECS系统中的实体数据转换为屏幕上的图像。渲染模块位于ECS与RHI之间，承接上层业务逻辑，抽象底层图形API。

### 1.2 模块职责

| 职责 | 描述 |
|------|------|
| 渲染管线管理 | 构建、执行渲染Pass序列 |
| SDF渲染 | 实现SDF Raymarching核心算法 |
| 光照计算 | 提供多光源照明模型 |
| 后处理效果 | 实现Bloom、AO等屏幕空间效果 |
| 材质系统 | 管理着色器与渲染状态 |
| 相机管理 | 提供多相机支持与视锥体裁剪 |

### 1.3 子系统划分

```
Rendering/
├── RHI/              # 渲染硬件接口抽象层
├── RenderGraph/      # 渲染管线管理系统
├── SDFRenderer/      # SDF Raymarching渲染器
├── Lighting/         # 光照系统
├── PostProcess/      # 后处理系统
└── Materials/        # 材质系统
```

## 2. RHI抽象层

### 2.1 设计目标

RHI（Render Hardware Interface）是渲染模块的底层抽象层：
- 统一接口，支持OpenGL/Vulkan/Metal等后端
- 游戏代码与具体图形API完全解耦
- 支持多线程渲染指令录制

### 2.2 核心接口

```cpp
/// 渲染设备抽象
class IRHIDevice {
public:
    virtual ~IRHIDevice() = default;
    virtual RHIPtr<IRHIBuffer> CreateBuffer(const BufferDesc& desc) = 0;
    virtual RHIPtr<IRHITexture> CreateTexture(const TextureDesc& desc) = 0;
    virtual RHIPtr<IRHIShader> CreateShader(const ShaderDesc& desc) = 0;
    virtual RHIPtr<IRHIPipeline> CreatePipeline(const PipelineDesc& desc) = 0;
    virtual RHICapabilities GetCapabilities() const = 0;
};

/// 命令列表抽象
class IRHICommandList {
public:
    virtual void BindPipeline(IRHIPipeline* pipeline) = 0;
    virtual void Draw(uint32_t vertexCount, uint32_t instanceCount, uint32_t firstVertex, uint32_t firstInstance) = 0;
    virtual void DrawIndexed(uint32_t indexCount, uint32_t instanceCount, uint32_t firstIndex, int32_t vertexOffset, uint32_t firstInstance) = 0;
    virtual void Dispatch(uint32_t groupCountX, uint32_t groupCountY, uint32_t groupCountZ) = 0;
};
```

### 2.3 后端实现

- OpenGL后端：优先实现
- Vulkan后端：预留接口

- 声明式渲染管线定义
- 自动资源管理

### 3.2 核心接口

### 4.6 Raymarching着色器

```glsl
float SceneSDF(vec3 p) {
    float d = kInfinity;
    for (int i = 0; i < objectCount; i++) {
        if ((objectPhaseMasks[i] & currentPhaseMask) != 0) {
            vec3 localP = InverseTransform(p, objectTransforms[i]);
            d = min(d, objectSDFs[i](localP));
        }
    }
    return d;
}

float RayMarch(vec3 ro, vec3 rd, float maxDist) {
    float t = 0.0;
    for (int i = 0; i < kMaxSteps; i++) {
        vec3 p = ro + rd * t;
        float d = SceneSDF(p);
        if (d < kSurfaceThreshold) return t;
        if (t > maxDist) break;
        t += d;
    }
    return -1.0;
}
```
