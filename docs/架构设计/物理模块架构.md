## 1. 模块概述

### 1.1 物理模块定位

Physics模块负责碰撞检测、相位物理和空间查询三大核心功能。基于SDF技术实现精确碰撞检测，与渲染模块共享SDF几何数据。

### 1.2 核心职责

| 职责 | 描述 |
|------|------|
| 碰撞检测 | 基于SDF的精确碰撞检测 |
| 相位物理 | 支持多相位空间切换的物理系统 |
| 空间查询 | 射线投射、球体查询、体积查询 |

## 3. 相位物理系统

### 3.1 核心概念

| 概念 | 定义 |
|------|------|
| Phase | 空间的一种状态 |
| PhaseMask | 相位位掩码 |
| PhaseRelation | 相位间碰撞规则 |

### 3.2 数据结构

```cpp
struct Phase {
    uint8_t id;
    std::string name;
    Vec3 color;
    bool allowCollision;
};

struct PhasePhysicsComponent {
    Phase currentPhase;
    PhaseMask availablePhases;
    PhaseMask collisionPhases;
    PhaseMask visiblePhases;
};
```

### 3.3 相位碰撞过滤

```cpp
class PhaseCollisionFilter {
public:
    static bool ShouldCollide(const PhasePhysicsComponent& a, const PhasePhysicsComponent& b, uint8_t currentPhase) {
        uint8_t phaseBit = (1 << currentPhase);
        bool aCollidable = (a.collisionPhases & phaseBit) != 0;
        bool bCollidable = (b.collisionPhases & phaseBit) != 0;
        return aCollidable && bCollidable;
    }
};
```
